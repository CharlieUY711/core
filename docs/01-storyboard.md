# CORE Executive Experience — Storyboard Completo

## Acto I — El Problema

### S00 · Hero (Apertura)
- Pantalla negra. "CORE" en el centro, grande, blanca.
- Sub: "The Operating System for Commerce"
- Fondo: red de nodos vectorial animada, apenas perceptible
- Scroll indicator discreto
- Intención: "Esto es serio. Quiero saber qué es."

### S01 · Fragmentación
- Afirmación: "You already have software."
- Nodos aparecen en scroll: CRM, ERP, Marketplace, Warehouse, Logistics, Customs, BI, Payments, Documents
- Líneas punteadas entre ellos — intentos fallidos de integración
- El caos crece progresivamente con el scroll
- Frase final: "The problem is not software. The problem is fragmentation."

### S02 · El Costo
- Duplicados: customers, products, inventory, processes
- Manual work, blind spots, disconnected decisions
- Frase: "Companies spend years integrating tools that were never designed to work together."

---

## Acto II — La Solución

### S03 · ¿Y si todo empezara conectado? (EL GIRO)
- El caos visual COLAPSA hacia el centro (animación de convergencia)
- Emerge estructura limpia: Entity · Product · Operation · Relationship · Document · Event
- Pregunta: "What if everything started connected?"
- Sub: "A single source of truth. Not another integration. A different starting point."
- ESTE ES EL MOMENTO MÁS IMPORTANTE DE TODA LA EXPERIENCIA

### S04 · CORE Foundation
- Diagrama orbital: Foundation en el centro, elementos orbitando
- Entities, Brands, Products, Customers, Documents, Operations, Inventory, Events
- Hover sobre cada elemento: micro-descripción
- "This is not a database. This is operational truth."

### S05 · El Ecosistema CORE
- 6 capacidades irradiando desde Foundation
- CORE Rep · Logistics · DirectShipment · Market · Finance · Intelligence
- Click en una capacidad: conexiones entre todas se iluminan
- Cada una: 2 líneas de valor de negocio, sin specs técnicas

## Acto III — La Evidencia

### S06 · Caso Paraguay → Uruguay
- Mapa SVG LATAM. Empresa en Paraguay.
- Before: lista de 7 pasos (estructura legal, logistics, warehouse, software, integrar, aduana, inventario, delivery)
- After: flujo CORE animado (Products → Free Zone → Inventory → Distribution → Market → Last Mile → Customer)
- Claim: "Operate in a new country without building a new operation."

### S07 · Caso Cardisan · LATAM
- Mapa LATAM. Cardisan en origen. Mercados destino iluminándose.
- 4 capacidades CORE activas sobre el mapa
- Claim: "Expand your market, not your complexity."

### S08 · Por qué CORE es diferente (MOMENTO TIPOGRÁFICO)
- Dos columnas:
  - Traditional: Many Systems / Many Providers / Many Integrations / Many Truths
  - CORE: One Entity / One Product / One Operation / One Truth
- "Many" aparece y acumula peso visual
- "One" barre todo — animación de transformación
- El momento visual más fuerte después del giro S03

### S09 · Alcance Global
- Mapa mundial SVG simplificado
- Arcos animados: Europa, Asia, Norte América → CORE → LATAM
- "CORE is born in South America. CORE is not limited to South America."
- "CORE becomes the operational gateway between global brands and regional markets."

### S10 · Visión 2035
- 4 contadores animados: 50,000 companies · 2M SKUs · 10 countries · 500M events/year
- "This is not a software product. This is infrastructure."
- CTA principal
